# PVAMU_REGISTRATION_SYSTEM_COMP3395

## Project Description
A database-driven system that implements priority-based waitlisting for full classes.

## Tech Stack
- MySQL
- XAMPP
- GitHub

## Database Setup Instructions

1. Start XAMPP (Apache + MySQL)
2. Open phpMyAdmin
3. Create database named: pvamu_registration
4. Import file: 127_0_0_1.sql

## Team Workflow

- Always pull latest changes before editing.
- After modifying database, export updated SQL file.
- Commit changes with clear message.
